# CLI команды управления

## Создание администратора
```bash
php create_admin.php
```

Интерактивно создает нового администратора или обновляет существующего пользователя.

**Пример:**
```
=== Создание администратора ===

Введите логин администратора: superadmin
Введите пароль: 
Подтвердите пароль: 

✓ Администратор 'superadmin' успешно создан!
```

---

## Отключение дефолтного admin
```bash
php disable_default_admin.php
```

Удаляет пользователя 'admin' (только если есть другие администраторы).

**Безопасность:** Скрипт не позволит удалить единственного администратора.

---

## Список пользователей
```bash
php list_users.php
```

Выводит список всех пользователей с их ролями.

**Пример вывода:**
```
=== Список пользователей ===

ID    Логин                Роль            Дата создания       
-----------------------------------------------------------------
1     admin                Администратор   30.01.2026 14:30    
2     john                 Доверенный      30.01.2026 15:45    
-----------------------------------------------------------------
Всего пользователей: 2
Администраторов: 1
```

---

## Изменение пароля
```bash
php change_password.php
```

Изменяет пароль любого пользователя.

**Пример:**
```
=== Изменение пароля пользователя ===

Введите логин пользователя: admin
Введите новый пароль: 
Подтвердите новый пароль: 

✓ Пароль пользователя 'admin' успешно изменен!
```

---

## Рекомендуемая последовательность для продакшена

1. Создать нового администратора:
```bash
   php create_admin.php
```

2. Проверить список пользователей:
```bash
   php list_users.php
```

3. Удалить дефолтного admin:
```bash
   php disable_default_admin.php
```

4. Убедиться, что новый администратор работает (войти через веб-интерфейс)

5. Удалить или ограничить доступ к CLI скриптам на продакшене
