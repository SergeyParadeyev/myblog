Ты — опытный PHP-разработчик.
Сгенерируй **полностью рабочее веб-приложение** для локального использования, похожее на блог.

⚠️ **Важно**:

* PHP **строго без ООП**, только процедурный стиль
* Никаких фреймворков
* Код должен быть простым, читаемым и реально запускаемым

---

## Функциональные требования

### Общее

* Приложение работает локально, но может быть опубликовано в интернет.
* Есть публичная и приватная части.
* Приватная часть доступна только после авторизации.

### Пользователи и права

* **Автор (1 аккаунт)**:
  * полный доступ (CRUD записей, категорий, комментариев).
* **Доверенные пользователи**:
  * чтение записей,
  * добавление комментариев.
* **Гости**:
  * только просмотр.

### Контент

* Записи
* Категории
* Хештеги
* Комментарии

### Файлы

* Загрузка файлов:

  * изображения,
  * видео,
  * документы,
  * архивы.

* Имена файлов должны быть неуникальными.
* В Markdown-записи можно вставлять:

  * загруженные файлы,
  * ссылки на них.

### Редактор

* Использовать **Markdown** для создания записей.
* Подсветка синтаксиса в блоках кода.
* Кнопка «Копировать код» у каждого блока.

## Технические требования

### Backend

* PHP (процедурный стиль).
* SQLite3 как база данных.
* Хранение сессий через PHP session.

### Frontend

* Bootstrap.
* jQuery.

---

## Требования к генерации кода

1. Генерируй код **пошагово, файл за файлом**.
2. В начале выведи **структуру каталогов проекта**.
3. Каждый файл:

   * выводи **целиком**,
   * в отдельном блоке кода,
   * с указанием имени файла.

4. Не используй псевдокод — только реальный PHP / HTML / JS.
5. Не сокращай код фразами вроде «здесь логика» или «и т.д.».
6. Используй только встроенные возможности PHP и SQLite.

---

## Минимально обязательные файлы

* `index.php` — публичная часть (список записей)
* `post.php` — просмотр записи и комментариев
* `login.php` / `logout.php`
* `admin/` — приватная часть
* `admin/post_create.php`
* `admin/post_edit.php`
* `admin/post_delete.php`
* `upload.php` — загрузка файлов
* `db/init.sql` — структура базы данных
* `db/db.php` — подключение к SQLite
